Javascript 101

Co je Javascript?

Javascript je programovací jazyk původně vytvořený pro skriptování webových stránek. Dnes je používán ve spoustu prostředí. Stránky, servery, machine learning, atd.

Zde se můžete naučit základy Javascriptu pro web.

Developer tools

Každý moderní prohlížeč má něco, čemu říkáme vývojářské nástroje, nebo "dev tools". Každý má trochu jinak vypadající ale mají prakticky stejnou funkčnost. Je možné je otevřít pomocí F12.

Zde přepneme na sekci "Konzole" a uvidíme náš Ahoj světe!

Proměnné

Proměnlivé jsou funkce v programování umožňující uložení dat během toho, co je program spuštěný. Například text, čísla, nebo pravda/lež (v pokročilejších případech třídy, ukazatele, paměťové adresy, atd.)
