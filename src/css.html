CSS 101


Vytváření CSS souboru pro stránku

Vytvořte nový soubor v našem projektu, pojmenujeme ho style.css.

Potom přidáme tento kus kódu do našeho index.html souboru.

Pravidla CSS

Představíme si nejdřív pravidla CSS. Tyto nám potom umožní porozumět rozložení a psaní našich stylů.

CSS kaskaduje

Je tím myšleno, že záleží na pořadí pravidel. Například v tomto kódu bude mít background-color větší prioritu než opacity


Specifikace

V CSS záleží, co stylujete. Protože některé věci mají větší prioritu, než jiné. List priority je tento:

  • - Inline styl
  • - ID attributy
  • - Třídy
  • - HTML elementy

Box model

Box model je nejkomplikovanější část CSS, takže se nebojte pokud ho hned nepochopíte.

Spočívá v tom, že každý HTML element má tento diagram.

Content reprezentuje samotný HTML element. Padding je malé místo těsně okolo contentu, který je průhledný. Border je hranice, která běží okolo contentu a paddingu. Margin je místo okolo všeho, je taky průhledný.


Teď konečně otevřeme style.css a vytvoříme si první CSS.

Prvně pojďme něco obarvit. Na toto využijeme background-color vlastnost. Vytvoříme CSS pravidlo ovlivňující html element a přidáme pravidlo do něj.

A když se podíváme na naši stránku, uvidíme že pozadí zhnědlo!


Lepší písmo

Dnes upřímně nikdo rád nevidí na stránce Times New Roman. Proto přidáme do naší stránky lepší font.

Vezmeme Roboto ze stránky fonts.google.com a přidáme tento kód do CSS souboru.

Teď jenom stačí abychom využili náš font a aplikovali ho na (například) <p> element

Třídy a IDčka

Tyto dvě věci se dají shrnout velmi jednoduše. Třídy a IDčka jsou způsoby jak stylovat individuální elementy, nebo skupiny elementů.

Rozdíl mezi IDčkem a třídou je, že třídy jsou používané na několika elementech. Například máme třídu pro 5 pixelový stín, který dáme na několik elementů a jeden element s description IDčkem.

V naší stránce využijeme třídy tím, že k našemu popisku Huga dáme ID description a dáme text-shadow vlastnost do CSS kódu.
